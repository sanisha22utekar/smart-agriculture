# Smart Agriculture Dashboard

A Flask-based Smart Agriculture Web Application with:

- Real-time Weather API
- Marathi Language Support
- Premium Dashboard UI
- Chart.js Analytics
- AWS EC2 Deployment Ready

## Technologies

- Python Flask
- Bootstrap 5
- Chart.js
- OpenWeatherMap API

## Run

```bash
pip install -r requirements.txt
python app.py
```

Open:
http://localhost:5000
